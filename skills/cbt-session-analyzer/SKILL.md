# Анализатор КПТ-сессий

Превращает транскрипты терапевтических сессий в структурированный отчёт по формату МИПЗ.

## Входные данные

TXT-файлы с транскриптами. Ожидаемый формат:

```
Спикер N:
HH:MM:SS — Текст реплики.
HH:MM:SS — Продолжение реплики.

Спикер M:
HH:MM:SS — Ответ другого участника.
```

Файлы нумеруются: `1.txt`, `2.txt`, ... (хронологический порядок сессий).

## Коррекция спикеров

**Важно:** Автоматическая диаризация часто ошибается — один человек может быть размечен как несколько разных спикеров.

**Алгоритм коррекции:**

1. **Идентификация по представлению** (начало первой сессии):
   - "Меня зовут [Имя]" — обычно терапевт представляется первым
   - Ответное имя — клиент

2. **Идентификация по роли:**

| Роль | Маркеры |
|------|---------|
| **Терапевт** | Задаёт вопросы, структурирует, резюмирует, назначает ДЗ, использует КПТ-терминологию |
| **Клиент** | Отвечает на вопросы, описывает ситуации и эмоции, рассказывает о себе |

3. **Объединение спикеров** — если несколько номеров явно принадлежат одному человеку → объединить

## Workflow

### 1. Подготовка

1. Получи все загруженные TXT-файлы
2. Отсортируй по номерам (1, 2, 3...)
3. Выполни коррекцию спикеров
4. Примени маппинг ко всем файлам

### 2. Анализ сессии

Для каждой сессии извлеки информацию по структуре отчёта (см. ниже).

Используй справочники:
- `references/cbt-techniques.md` — для распознавания техник
- `references/cognitive-distortions.md` — для выявления искажений
- `references/conceptualization-guide.md` — для структуры концептуализации

## Критерии отбора цитат

Выноси в раздел "Цитаты из сессии":
- Формулировки глубинных/промежуточных убеждений
- Яркие примеры когнитивных искажений
- Инсайты клиента
- Примеры эффективных интервенций терапевта
- Эмоционально значимые моменты

## Оценка эффективности техник

| Оценка | Критерии |
|--------|----------|
| **Высокая** | Клиент получил инсайт, сформулировал альтернативу, снизилась интенсивность эмоции |
| **Средняя** | Частичное продвижение, требуется продолжение работы |
| **Низкая** | Техника не достигла цели, клиент не включился, требуется другой подход |

## Ограничения

- Качество анализа зависит от качества транскрипции
- Невербальные сигналы недоступны
- Анализ — для рефлексии и супервизии, не замена клинической оценки
