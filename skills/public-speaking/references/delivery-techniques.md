# Техники подачи выступления

Рекомендации по невербалике, интерактиву, визуализации и репетиции.

## Невербальная коммуникация

### Голос

| Элемент | Как использовать |
|---------|-----------------|
| **Громкость** | Говорить чуть громче, чем кажется комфортным. В большом зале — ещё громче. |
| **Темп** | Замедляться на ключевых мыслях. Обычный темп — 120-150 слов/мин. |
| **Паузы** | Перед важным — пауза 2-3 сек. После важного — тоже. Не бояться тишины. |
| **Интонация** | Не монотонно. Вопросы — вверх. Утверждения — вниз. Акценты — выделять. |
| **Артикуляция** | Чётко. Особенно окончания слов. |

### Тело

| Элемент | Рекомендация |
|---------|--------------|
| **Поза** | Устойчивая, ноги на ширине плеч. Не переминаться. |
| **Руки** | Жесты от плеч, не от локтей. Открытые ладони. Не скрещивать. |
| **Перемещение** | Осознанное. К аудитории — при важном. В сторону — при переходе. |
| **Зрительный контакт** | 3-5 секунд на человека. Охватывать весь зал. Не смотреть поверх голов. |
| **Мимика** | Соответствует содержанию. Улыбка уместна, но не постоянная. |

### Что избегать

- ❌ Руки в карманах (закрытость)
- ❌ Скрещённые руки (защита)
- ❌ Раскачивание (нервозность)
- ❌ "Эээ", "нуу", слова-паразиты
- ❌ Взгляд в пол/потолок/экран
- ❌ Чтение со слайдов спиной к залу

## Интерактивность

### Типы интерактива

| Тип | Когда | Как |
|-----|-------|-----|
| **Риторический вопрос** | Всегда уместен | "Знакомо?" — пауза 2 сек |
| **Поднятие рук** | Разогрев, вовлечение | "Поднимите руку, кто сталкивался с..." |
| **Мини-опрос** | Выяснить контекст | "Сколько человек работает удалённо?" |
| **Обращение к конкретному** | С осторожностью | Только если уверен в реакции |
| **Обсуждение с соседом** | Длинные сессии | "30 секунд — обсудите с соседом" |
| **Практическое задание** | Воркшопы | "Запишите три идеи..." |

### Правила интерактива

1. Предупреждать об интерактиве ("Сейчас я задам вопрос...")
2. Давать время на реакцию (минимум 3 секунды)
3. Благодарить за участие
4. Не осуждать "неправильные" ответы
5. Связывать ответы с содержанием

### Работа с вопросами

| Ситуация | Действие |
|----------|----------|
| Вопрос понятен | Ответить лаконично, связать с темой |
| Вопрос непонятен | Переспросить, уточнить |
| Провокация | Спокойно, не оправдываться, переформулировать |
| Не знаешь ответ | "Хороший вопрос. Давайте обсудим после" |
| Затянувшийся диалог | "Это большая тема, обсудим в перерыве" |

## Визуализация

### Принципы слайдов

| Принцип | Что значит |
|---------|-----------|
| **Один слайд — одна мысль** | Не перегружать |
| **Минимум текста** | Ключевые слова, не предложения |
| **Крупный шрифт** | Читаемо с последнего ряда |
| **Контраст** | Тёмный фон — светлый текст или наоборот |
| **Изображения > текст** | Картинка говорит больше |
| **Данные визуально** | Графики вместо таблиц |

### Что показывать

| Контент | Как визуализировать |
|---------|-------------------|
| Структура | Нумерованный список (но не зачитывать!) |
| Данные | График, диаграмма |
| Процесс | Схема, стрелки |
| Сравнение | Два столбца, до/после |
| Эмоция | Сильное фото |
| Цитата | Крупно, с атрибуцией |

### Антипаттерны

- ❌ Слайды как суфлёр (много текста, чтение)
- ❌ Сложные таблицы мелким шрифтом
- ❌ Анимация ради анимации
- ❌ Слайд на каждые 30 секунд
- ❌ Чтение того, что на экране

## Работа с волнением

### До выступления

| Техника | Как |
|---------|-----|
| Дыхание | 4-7-8: вдох 4 сек, задержка 7, выдох 8 |
| Физика | Потрясти руками, попрыгать, размять плечи |
| Позитивная визуализация | Представить успешное выступление |
| Приход заранее | Освоиться в пространстве |
| Проверка техники | Исключить технические сюрпризы |

### Во время выступления

- Волнение — это нормально, даже полезно (энергия)
- Начать с заученного — первые 30 секунд наизусть
- Найти доброжелательные лица
- Пауза — не страшно, а мощно
- Если сбился — не извиняться, продолжить

## Репетиция

### Зачем репетировать

- Проверить хронометраж (реально дольше, чем кажется)
- Отловить слабые переходы
- Запомнить ключевые точки
- Снизить тревогу

### Как репетировать

| Этап | Что делать |
|------|-----------|
| 1. Читка | Прочитать вслух, отметить сложные места |
| 2. Структура | Пройти по тезисам без текста |
| 3. Полный прогон | С таймером, стоя, вслух |
| 4. Перед зеркалом/камерой | Отследить невербалику |
| 5. Перед другом | Получить обратную связь |

### Что репетировать особенно

- Первые 60 секунд (снять тревогу старта)
- Переходы между тезисами
- Ключевые фразы (должны звучать естественно)
- Финал (уверенное завершение)

## Чек-лист перед выступлением

**За день:**
- [ ] Полный прогон с таймером
- [ ] Слайды проверены на чужом устройстве
- [ ] Резервная копия презентации

**За час:**
- [ ] Проверить технику на месте
- [ ] Пройтись по пространству
- [ ] Попить воды (но не перед самым выходом)
- [ ] Дыхательные упражнения

**За минуту:**
- [ ] Выключить телефон
- [ ] Улыбнуться
- [ ] Вспомнить первую фразу
