# Вопросы для сбора требований

Структурированный список вопросов для глубокого понимания задачи перед созданием промта.

## Уровень 1: Обязательные (задай всегда)

### Цель
- "Какую конечную задачу должен решать ассистент?"
- "Что считается успешным результатом работы?"
- "Какую трансформацию входа в выход обеспечивает ассистент?"

### Аудитория
- "Кто будет использовать ассистента? (новички / специалисты / смешанная)"
- "Какой уровень знаний в предметной области у пользователей?"

---

## Уровень 2: Важные (задай при возможности)

### Контекст использования
- "В каких ситуациях будет вызываться ассистент?"
- "Какой уровень срочности типичных запросов?"
- "Есть ли интеграция с другими системами/процессами?"

### Предметная область
- "Какие ключевые концепции и термины должен знать ассистент?"
- "Есть ли специфические методологии или стандарты?"
- "Какие смежные области могут потребоваться?"

### Формат
- "Какая структура ответов наиболее полезна?"
- "Нужна ли стандартизация формата?"
- "Есть ли примеры желаемого результата?"

---

## Уровень 3: Уточняющие (при необходимости)

### Стиль
- "Какой тон предпочтителен? (формальный / дружелюбный / нейтральный)"
- "Какой уровень детализации оптимален?"
- "Нужна ли эмоциональная составляющая?"

### Ограничения
- "Чего ассистент НЕ должен делать?"
- "Какие темы или подходы исключены?"
- "Есть ли ограничения по объёму ответов?"

### Граничные случаи
- "Как обрабатывать неоднозначные запросы?"
- "Что делать при недостатке информации?"
- "Как реагировать на запросы вне компетенции?"

---

## Быстрые шаблоны вопросов

### Для технических ассистентов
1. Какой стек/технологии?
2. Уровень пользователей (junior/middle/senior)?
3. Нужен ли рабочий код или концептуальные объяснения?
4. Есть ли code style guide?

### Для консультативных ассистентов
1. Какая область консультирования?
2. Глубина анализа (быстрый совет vs глубокий разбор)?
3. Нужны ли источники/обоснования?
4. Уровень директивности (советы vs направляющие вопросы)?

### Для творческих ассистентов
1. Жанр/формат контента?
2. Целевая аудитория контента?
3. Tone of voice бренда?
4. Есть ли примеры желаемого стиля?

### Для аналитических ассистентов
1. Тип данных/информации для анализа?
2. Какие выводы нужны (описательные / предписывающие)?
3. Формат отчёта?
4. Уровень уверенности в выводах?

---

## Признаки недостаточного брифа

⚠️ Нужно уточнить, если:
- Цель описана абстрактно ("помогает с X")
- Нет критериев успеха
- Аудитория не определена
- Нет примеров желаемого результата
- Противоречивые требования
