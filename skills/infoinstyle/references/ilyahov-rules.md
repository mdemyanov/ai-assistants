# Правила инфостиля

Адаптация методологии Ильяхова для редактирования ИИ-текстов.

---

## 1. Стоп-слова

Слова, которые можно удалить без потери смысла.

### Усилители (удалить)

| Убрать | Почему |
|--------|--------|
| очень | не добавляет информации |
| крайне | преувеличение |
| чрезвычайно | канцелярит |
| исключительно | оценка без факта |
| максимально | размыто |
| достаточно | неконкретно |

### Вводные (удалить или заменить на факт)

| Убрать | Альтернатива |
|--------|--------------|
| конечно | [удалить] |
| безусловно | [удалить] |
| несомненно | [удалить] |
| как известно | [удалить] |
| очевидно, что | [удалить] |
| следует отметить | [удалить] |

### Размытые (уточнить)

| Размытое | Заменить на |
|----------|-------------|
| какой-то | конкретное название |
| некоторый | число или процент |
| определённый | какой именно |
| различные | перечислить |
| многие | число или «большинство» |
| ряд | число |

## 2. Канцелярит → Простой язык

| Канцелярит | Простой язык |
|------------|--------------|
| осуществлять | делать |
| производить | делать |
| реализовывать | делать / внедрять |
| в рамках | при / когда / в |
| на основании | по |
| в целях | чтобы |
| посредством | через |
| в связи с | из-за |
| в случае если | если |
| в настоящее время | сейчас |
| на данный момент | сейчас |
| в ближайшее время | скоро / до [дата] |
| является | это / [тире] |
| представляет собой | это / [тире] |
| выступает в качестве | это / работает как |
| обеспечивать | давать / гарантировать |
| способствовать | помогать |

---

## 3. Пассив → Актив

**Формула:** [Кто] + [глагол] + [что]

| Пассив | Актив |
|--------|-------|
| Было принято решение | Мы решили |
| Проводится анализ | Аналитики изучают |
| Будет рассмотрено | Рассмотрим |
| Планируется внедрение | Внедрим |
| Осуществляется контроль | Контролируем |

**Исключение:** Пассив уместен, когда субъект неизвестен или неважен.

---

## 4. Отглагольные → Глаголы

| Отглагольное | Глагол |
|--------------|--------|
| осуществление | делать |
| проведение | провести |
| обеспечение | обеспечить |
| реализация | внедрить |
| оптимизация | оптимизировать |
| использование | использовать |
| формирование | сформировать |

---

## 5. Пирамида

Структура текста: главное → детали.

```
[Вывод / главная мысль]     ← первое предложение
[Ключевые аргументы]        ← 2-3 факта
[Детали для интересующихся] ← остальное
```

**Тест:** Если удалить последний абзац — смысл сохранится?

---

## 6. Сильные глаголы

Заменяй слабые связки на действия.

| Слабо | Сильно |
|-------|--------|
| является важным | влияет / определяет |
| имеет значение | важен для |
| оказывает влияние | влияет |
| принимает участие | участвует |
| даёт возможность | позволяет |
| ведёт к увеличению | увеличивает |

---

## 7. Конкретика

| Абстракция | Конкретика |
|------------|------------|
| значительно выросло | выросло на 23% |
| многие клиенты | 78% клиентов |
| в ближайшее время | до 15 февраля |
| существенная экономия | экономия 50 000 руб. |
| высокая эффективность | решает задачу за 2 часа вместо 8 |

---

## Чеклист редактирования

1. [ ] Главное в первом предложении?
2. [ ] Нет предложений длиннее 2 строк?
3. [ ] Нет канцелярита?
4. [ ] Нет стоп-слов?
5. [ ] Пассив заменён на актив?
6. [ ] Есть конкретика (цифры, даты, имена)?
7. [ ] Абстракции подкреплены примерами?
8. [ ] Тон соответствует аудитории?
9. [ ] Можно удалить последний абзац без потери смысла?
10. [ ] Авторский голос сохранён?
