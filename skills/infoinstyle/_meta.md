# Инфо в стиль (InfoInStyle)

Skill для адаптации ИИ-текстов под инфостиль.

## Описание

Убирает характерные паттерны машинно-сгенерированных текстов и применяет правила инфостиля:
- Диагностика ИИ-паттернов по 4 категориям (Content, Language, Style, Communication)
- Очистка от канцелярита и стоп-слов
- Усиление конкретикой (факты, цифры, примеры)
- Три режима работы (quick/standard/deep)

## Компоненты

- `SKILL.md` — workflow и формат ответа
- `references/ai-patterns.md` — 20 паттернов ИИ-текста с примерами
- `references/ilyahov-rules.md` — правила инфостиля (адаптация Ильяхова)
- `assets/examples-before-after.md` — примеры редактирования

## Триггеры использования

- Отредактировать текст / почистить текст
- Убрать ИИ-паттерны / убрать роботность
- Переписать в инфостиле / по Ильяхову
- Сделать текст человечнее / живее
- Убрать воду / канцелярит / штампы
- Добавить конкретики

## Связи

- Методология: Максим Ильяхов, «Пиши, сокращай», «Ясно, понятно»
- Референс: humanizer (blader/humanizer)
