# meeting-debrief

Постобработка встреч — извлечение результатов и обновление vault.

## Описание

Анализирует встречу и формирует предложения по обновлению vault:
- Извлекает договорённости, решения, задачи
- Сравнивает с повесткой (план vs факт)
- Обновляет profile.md, создаёт заметки
- Задаёт уточняющие вопросы при необходимости

## Триггеры

- «обработай встречу»
- «разбери транскрипт»
- «что решили на встрече»
- «зафиксируй результаты»
- «debrief по [встрече]»

## Источники

- Транскрипт (.txt из Контур Толк и др.)
- Переписка (текст из чата/почты)
- Интервью (диалог с пользователем)
- Комбинация источников
