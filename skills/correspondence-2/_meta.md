# Correspondence 2.0

Skill для создания деловой переписки по методологии Саши Карепиной.

## Описание

Помогает создавать эффективные деловые письма и сообщения с учётом:
- Формата коммуникации (email vs мессенджер)
- Цели сообщения (информировать, убедить, решить проблему)
- Уровня формальности и отношений с адресатом

## Компоненты

- `SKILL.md` — основные инструкции
- `references/karepina-method.md` — детали методологии "Переписка 2.0"

## Триггеры использования

- Написать письмо / email
- Написать сообщение в чат / мессенджер
- Ответить на переписку
- Составить деловое сообщение

## Связи

- Методология: Саша Карепина, "Переписка 2.0"
