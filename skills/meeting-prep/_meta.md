# meeting-prep

Подготовка к встречам на основе данных vault.

## Описание

Создаёт файл повестки, автоматически собирая:
- Договорённости из профилей
- Вопросы для 1-1
- Контекст из проектов
- Insights из анализов (deep-режим)

## Триггеры

- «подготовь к встрече»
- «готовлюсь к 1-1»
- «нужна повестка»
- «agenda для [встречи]»

## Компоненты

- `SKILL.md` — основная логика
- `references/data-sources.md` — карта данных vault
- `references/meeting-types.md` — специфика типов встреч
